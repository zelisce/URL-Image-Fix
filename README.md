
# Server List Endstone Plugin

A simple plugin for Endstone that allows Script API to use URL images inside forms.

## Installation

1. Download the latest release
2. Place them in your server's `plugins` directory
3. Restart your server

## Features

- Display URL images inside forms by using Script API
